# Code of Conduct

## Professional Expectations

This project is used to test security tooling behavior. Keep interactions technical, respectful, and evidence-driven.

## Expected Behavior

- Be precise and professional
- Critique ideas and implementations, not people
- Share reproducible evidence when reporting issues

## Unacceptable Behavior

- Harassment, abuse, or personal attacks
- Intentionally misleading security advice
- Posting secrets or sensitive tenant data
- Spam and repeated low-effort noise

## Enforcement

Maintainers may edit, lock, remove, or restrict participation that harms project quality or safety.

Security concerns: security@vyrox.security
